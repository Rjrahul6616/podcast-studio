# 🎙️ Podcast Studio Pro

Generate professional podcast scripts from any PDF and produce high‑quality audio with distinct AI voices – all for free!

## ✨ Features

- 📄 Extract text from PDFs and edit it inline
- 🔍 Automatically highlights important keywords
- 📌 Select topics from the document or add your own
- 🤖 Multiple free AI models via OpenRouter (DeepSeek, Gemini, Llama, etc.)
- 🎭 Choose a script style: Casual, Funny, Formal, etc.
- ✏️ Modify the generated script with natural language instructions
- 🔊 Generate a podcast audio file with separate male/female voices
- 🎨 Beautiful, animated speaker indicator
- 🔒 Offline TTS (pyttsx3) – no internet required for voice generation
